# EcoGuardian Alert

Sistema de alertas descentralizado para defensores ambientales en zonas de montaña, basado en Briar.

## Funcionalidades
- Envío de alertas de peligro o SOS sin internet (Bluetooth/WiFi Direct)
- Red en malla entre dispositivos autorizados
- Cifrado de extremo a extremo
- Funciona en zonas sin cobertura móvil

## Tecnología
- Motor de comunicación: [Briar](https://briarproject.org/) (GPL-3.0)
- Interfaz: Android nativo (Kotlin)

## Instalación
1. Compila con Android Studio o genera APK mediante GitHub Actions.
2. Los usuarios deben escanear códigos QR para establecer contactos.
